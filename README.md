# Getting Started
In order to get this project up and running you will need to follow the next steps.

### Prerequisites

* NodeJS >=18

### Setup

Install dependencies
```
npm install
```
Build(Not required)
```
npm run build
```
Run as Dev
```
npm run dev
```
